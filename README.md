# Generador de memes simple

Genera [memes] fácil y rápido guardando en localstorage

## Tech

- React
- JS

## Instalación

Se requiere nodejs y npm

Instalar dependencias.

```sh
cd meme-gen-localst
npm i
npm run dev
```
